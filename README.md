# EEG-based Prediction of Learning Outcomes

Project for the Machine Learning course (Temple University).

- Task: predict student understanding of online lecture from EEG band-power features.
- Data: public Kaggle dataset "EEG data / Distance learning environment".
- Code: three Jupyter notebooks (`1_EDA.ipynb`, `2_Modeling.ipynb`, `3_ClassImbalance_FNN_XGBoost.ipynb`).
- Report: `final_project_report.pdf`.
- Environment: `requirements.txt`.
